# lizpat

A lisp written in zig

## Pseudocode

\* means it is a stretch goal/deferred

[ ]
read(str) -> s-expr
 - Reads in a string

[ ]
eval(s-expr) -> s-expr
 - Evaluates an s-expression

[ ]
print(s-expr) -> void 
 - prints in human readable format

[ ]
prn(s-expr)* -> void 
 - imperatively prints in *read* readable format
